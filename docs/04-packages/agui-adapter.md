# Package: agui-adapter

**`@open-design/agui-adapter`** — AGUI protocol adapter.

## Purpose

Adapts the AGUI (Agent UI) protocol for use with Open Design's agent runtimes. Translates between AGUI stream events and the daemon's internal event format.

## Consumers

- `@open-design/daemon` — uses for AGUI-compatible agent runs

## Commands

```bash
pnpm --filter @open-design/agui-adapter typecheck
```
