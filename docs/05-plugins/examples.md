## Using an Example Plugin

1. Browse examples in the web UI's Plugins/Examples section
2. Each example shows a preview of what it produces
3. Select one and apply it to your project
4. It pre-fills the composer with the plugin's pipeline
5. Send your prompt — the agent follows the plugin's workflow

## Using an Example as a Starting Point

1. Copy the example folder from `plugins/_official/examples/<plugin-id>/`
2. Place your copy in `plugins/spec/examples/<your-plugin-id>/` or a new community plugin folder
3. Modify `SKILL.md` for your use case
4. Update `open-design.json` with new name, version, pipeline, and inputs
5. Validate and test
